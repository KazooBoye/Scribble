# Resources for Scribble Pygame Client

## Required Image Assets

Place the following PNG files in this directory:

### Icons (32x32 pixels, transparent background)
- `avatar_default.png` - Simple person silhouette icon (used when no specific avatar)
- `icon_online.png` - Green circle or checkmark (10x10px)
- `icon_offline.png` - Red circle or X (10x10px)
- `icon_drawing.png` - Pencil or brush icon (16x16px)
- `icon_clear.png` - Eraser or trash icon (24x24px)
- `icon_send.png` - Send arrow or paper plane (20x20px)

### Optional (the app will work without these)
- `bg_wood.jpg` - Wooden texture for background (800x600px or tileable)
- `logo.png` - Game logo (200x100px)

## Creating Simple Icons

You can create these using any image editor or generate them programmatically.

### Quick Method (No Image Editor Needed):
The game will auto-generate simple colored circles if images are missing.

### Using Online Tools:
- https://www.flaticon.com/ - Free icons (attribution required)
- https://icons8.com/ - Free icons
- https://www.canva.com/ - Create custom icons

## File Format
- **Format**: PNG with transparency
- **Color Mode**: RGBA
- **Background**: Transparent
